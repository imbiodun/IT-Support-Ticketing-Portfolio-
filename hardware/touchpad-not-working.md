# Issue Summary
User reports that the laptop touchpad is not responding or stops working intermittently.

# Environment
- Windows 10 laptop
- Built‑in touchpad

# User Description
"My touchpad suddenly stopped working. I can only move the cursor with an external mouse."

# Troubleshooting Steps
- Restarted the laptop to rule out temporary system glitches
- Checked if the touchpad was accidentally disabled using the function key (e.g., Fn + F5/F7 depending on model)
- Verified touchpad settings in **Settings → Devices → Touchpad**
- Ensured that “Touchpad” toggle was turned ON
- Checked Device Manager for touchpad driver status
- Uninstalled and reinstalled the touchpad driver
- Updated the driver using Windows Update
- Tested touchpad functionality in BIOS to confirm hardware status

# Resolution
Re-enabling the touchpad in Windows settings restored full functionality.

# Root Cause
Touchpad was accidentally disabled through system settings or keyboard shortcut.

# Notes
Advised user to avoid pressing function keys that toggle touchpad settings unintentionally.
