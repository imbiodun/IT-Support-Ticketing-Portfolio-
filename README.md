# IT Support Ticketing Portfolio

This portfolio showcases my hands-on experience with troubleshooting, documentation, and IT support workflows. 
Each entry is written in a real help-desk ticket format, including issue summaries, troubleshooting steps, resolutions, and notes.

The goal of this portfolio is to demonstrate:
- Practical problem-solving skills
- Ability to document issues clearly and professionally
- Familiarity with Windows, Microsoft 365, networking, virtualization, and hardware troubleshooting
- Understanding of help-desk workflows and structured ticketing
- Ability to communicate technical issues in a clear and user-friendly way
