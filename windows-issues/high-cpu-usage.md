# Issue Summary
Laptop experiencing slow performance due to consistently high CPU usage.

# Environment
- Windows 11 laptop
- Task Manager used for diagnostics

# User Description
"My laptop is very slow and keeps freezing. I checked Task Manager and the CPU is almost always above 90%."

# Troubleshooting Steps
- Opened Task Manager to confirm high CPU usage
- Sorted processes by CPU to identify the top-consuming application
- Checked if the process was a background app, browser tab, or system process
- Closed unnecessary applications
- Restarted the laptop to clear stuck background processes
- Checked for pending Windows updates
- Scanned for malware using Windows Security
- Verified that no startup apps were consuming excessive resources

# Resolution
Closing unnecessary background applications and restarting the device reduced CPU usage to normal levels.

# Root Cause
A background application was consuming excessive CPU resources.
