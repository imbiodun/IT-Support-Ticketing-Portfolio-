# Issue Summary

User unable to connect to Wi-Fi. Connection keeps failing.



# Environment

* Windows 10 laptop
* Home Wi-Fi network



# User Description

"Wi-Fi was working before, but now it keeps disconnecting and won’t reconnect."



# Troubleshooting Steps

* Confirmed Wi-Fi password with the user
* Turned Wi-Fi off and on again
* Restarted the laptop
* Removed the Wi-Fi network and reconnected
* Checked if other devices could connect to the same network
* Verified that the router was working



# Resolution

Reconnecting to the Wi-Fi after removing the saved network resolved the issue.



# Root Cause

Incorrect or outdated saved Wi-Fi credentials.



# Notes

Advised user to reconnect manually if the issue happens again.
